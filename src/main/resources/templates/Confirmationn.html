Confirmation de votre rendez-vous

Voici les étapes de votre prise de rendez-vous :

1. Choisir votre créneau horaire

Vous pouvez choisir un créneau horaire qui vous convient pour votre consultation.

2. Remplir vos informations

Veuillez renseigner vos informations personnelles pour confirmer votre rendez-vous.

3. Confirmation de votre rendez-vous

Une fois toutes les informations remplies, vous recevrez une confirmation par email.
